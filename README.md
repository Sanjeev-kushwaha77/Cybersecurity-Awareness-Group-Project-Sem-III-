Cybersecurity Awareness Survey Among Students
Project Overview

This is a college group website project focused on spreading cybersecurity awareness among students.

The main objective of this project is to understand how aware students are about cyber threats and to educate them on how to stay safe from:

Phishing attacks

Malware

Social engineering

Password attacks

Unsafe downloads

Online fraud

This project was completely planned and executed by our team at the ground level.

Methodology
1. Awareness Campaign

We interacted directly with students and explained:

How phishing scams work

How malware spreads

Risks of sharing OTP and passwords

Importance of strong passwords

Safe internet browsing habits

2. Data Collection

We created a Google Form survey and collected real responses from students.

The survey included questions about:

Password practices

Public Wi-Fi usage

Antivirus usage

Link clicking behaviour

Awareness of phishing and malware

3. Real-Time Dashboard

The collected data is displayed on our website through a live dashboard that shows:

Charts and graphs

Awareness percentage

Risk behaviour patterns

Security knowledge level

Outcome

Through this project, we:

Identified gaps in cybersecurity awareness

Helped students understand online threats

Promoted safe digital practices

Encouraged responsible internet usage

Conclusion

The survey revealed that while students frequently use the internet, many lack proper knowledge about cybersecurity risks.

This project was an effort to bridge that gap by combining awareness campaigns, real data analysis, and practical education.
